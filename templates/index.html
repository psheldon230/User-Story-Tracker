Jira CSV → Excel sync

How it works — same machine

  1. In Novartis Jira → run your filter → Export → CSV
  2. Download your Excel tracker from SharePoint
  3. Upload both files below and click Sync
  4. Download the updated Excel → upload back to SharePoint

VDI Step — open this page in GitHub Codespaces

  1. On VDI: open GitHub → go to the repo → Code → Codespaces → Open
  2. In the Codespace terminal run: python app.py → open the public URL
  3. In Novartis Jira → run your filter → Export → CSV
  4. Upload the CSV here and click Parse & Download JSON
  5. Send the .json file to yourself over Teams
  6. Switch to your Deloitte machine → open localhost:5000 → use Office: Sync JSON

Deloitte Machine Step — localhost:5000

  1. Download the .json file from Teams (sent from VDI)
  2. Download your Excel tracker from SharePoint
  3. Upload both files below and click Sync Stories
  4. Download the updated Excel → upload back to SharePoint
