Profile → Personal Access Tokens → Create token How do I get one?

Getting a Jira Server Personal Access Token:

  1. Open Jira in your browser and log in
  2. Click your profile picture (top-right corner)
  3. Select Profile from the dropdown
  4. In the left sidebar click Personal Access Tokens
  5. Click Create token, give it a name, set an expiry
  6. Click Create and copy the token immediately — it won't be shown again
  7. Paste it into the field above

⚠️ PATs require Jira Server 8.14 or later. If your Jira is older, use Username / Password instead.
