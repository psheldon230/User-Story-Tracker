Jira CSV → Excel sync

How it works — same machine

  1. In Novartis Jira → run your filter → Export → CSV
  2. Download your Excel tracker from SharePoint
  3. Upload both files below and click Sync
  4. Download the updated Excel → upload back to SharePoint

VDI Step — use this tab on VDI

  1. In Novartis Jira → run your filter → Export → CSV
  2. Upload the CSV here and click Parse & Download JSON
  3. Save the downloaded .json file and send it to yourself over Teams
  4. Switch to your Deloitte machine → use Sync JSON with Functional Testing tab

Deloitte Machine Step

  1. Download the .json file from Teams (sent from VDI)
  2. Download your Excel tracker from SharePoint
  3. Upload both files below and click Sync Stories
  4. Download the updated Excel → upload back to SharePoint
